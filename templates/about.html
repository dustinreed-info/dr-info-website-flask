About This Website

Built with modern web technologies and deployed on AWS infrastructure

This website was developed using Python and the Flask web framework, showcasing modern web development practices with responsive design and clean architecture. The entire site is built with semantic HTML5, modern CSS with CSS Grid and Flexbox, and follows accessibility best practices.

I've implemented a robust CI/CD pipeline that automatically deploys the latest version of my website by monitoring changes to my GitHub repository. The deployment is handled by AWS Elastic Beanstalk, which provides rolling updates with an additional batch to ensure high availability and zero downtime.

To further enhance availability and performance, I've set up EC2 instances across multiple availability zones behind an Application Load Balancer. The auto-scaling policy dynamically adjusts the number of instances based on current resource demand, ensuring efficient resource utilization and cost optimization.
